# ライブラリ・パッケージ・モジュール・メソッド

## ライブラリとは

ライブラリとは、便利な機能をまとめたもの。

Pythonでは、音声処理や画像処理など、さまざまなライブラリを利用できる。

例

* pydub（音声処理）（パイハブ）
* Pillow（画像処理）（ピロー）

---

## パッケージとは

パッケージとは、関連するモジュールをまとめたもの。

例

```python
from PIL import Image
```

この場合、

* `PIL` がパッケージ
* `Image` がモジュールです。

---

## モジュールとは

モジュールとは、ひとまとまりの機能を持った部品。

画像を扱う `Image` や、音声を扱う `AudioSegment` などがあります。

---

## メソッドとは

メソッドとは、モジュールやオブジェクトが持っている機能（命令）。

例

```python
Image.open("photo.jpg")
```

この場合、

`open()` がメソッドです。

画像を開く機能を実行しています。

---

## ドット（`.`）の意味

`.` は、

* 「〜の」
* 「〜にある」

というイメージで考えると分かりやすい。

例

```python
Image.open("photo.jpg")
```

これは、

> Image の open メソッドを使う

という意味になります。

同様に、

```python
AudioSegment.from_wav("test.wav")
```

は、

> AudioSegment の from_wav メソッドを使う

という意味になる。

---

## 変数

変数は、データを保存するための名前です。

例

```python
sound = AudioSegment.from_wav("test.wav")
```

`sound` は変数です。

変数名は自由に付けることができます。

例えば、

```python
a = AudioSegment.from_wav("test.wav")
```

でも動作します。

ただし、`sound` のように意味が分かる名前を付けた方が、後でコードを読んだときに理解しやすくなります。

---

## 今回学んだこと

* ライブラリは便利な機能をまとめたもの
* パッケージはモジュールをまとめたもの
* モジュールは機能のまとまり
* メソッドは機能（命令）
* `.` は「〜の」「〜にある」というイメージ
* 変数名は自由だが、意味のある名前を付けると読みやすい
