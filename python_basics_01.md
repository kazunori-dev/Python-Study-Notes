# ライブラリ・パッケージ・モジュール・クラス・メソッド

## ライブラリとは

ライブラリとは、便利な機能をまとめたものです。

Pythonでは、音声処理や画像処理など、さまざまなライブラリを利用できます。

例

* `pydub`（音声処理）
* `Pillow`（画像処理）

---

## パッケージとは

パッケージとは、関連するモジュールをまとめたものです。

例

```python
from PIL import Image
```

この場合、

* `PIL` がパッケージ
* `Image` がモジュール

です。

---

## モジュールとは

モジュールとは、ひとまとまりの機能を持ったPythonファイルです。

例

```python
from pydub import AudioSegment
```

この場合、

* `pydub` の中に `AudioSegment` というクラスが定義されています。

---

## クラスとは

クラスとは、データと機能（メソッド）をまとめた設計図です。

学校で例えると、

クラスには先生や生徒がいて、それぞれ役割があります。

Pythonでも、クラスには役割や機能がまとめられています。

例えば、

```python
AudioSegment.from_wav("test.wav")
```

この `AudioSegment` がクラスです。

このクラスには、

* `from_wav()`
* `from_mp3()`
* `export()`

など、音声を扱うためのメソッドが用意されています。

---

## オブジェクトとは

オブジェクトとは、クラス（設計図）から作られた実際のデータです。

例

```python
sound = AudioSegment.from_wav("test.wav")
```

この場合、

* `AudioSegment` はクラス
* `sound` はオブジェクトです。

`AudioSegment`という設計図から、`sound`という実際に操作できる音声データが作られています。

---

## メソッドとは

メソッドとは、クラスやオブジェクトが持っている機能（命令）のことです。

例

```python
Image.open("photo.jpg")
```

この場合、

`open()` がメソッドです。

同様に、

```python
AudioSegment.from_wav("test.wav")
```

では、

`from_wav()` がメソッドになります。

---

## ドット（`.`）の意味

`.` は、

* 「〜の」
* 「〜にある」

というイメージで考えると分かりやすいです。

例

```python
Image.open("photo.jpg")
```

これは、

> Image の open メソッドを使う

という意味です。

同様に、

```python
AudioSegment.from_wav("test.wav")
```

は、

> AudioSegment の from_wav メソッドを使う

という意味になります。

---

## 変数

変数は、データを保存するための名前です。

例

```python
sound = AudioSegment.from_wav("test.wav")
```

`sound` は変数です。

変数名は自由に付けることができます。

例えば、

```python
a = AudioSegment.from_wav("test.wav")
```

でも動作します。

ただし、`sound` のように意味が分かる名前を付けた方が、後でコードを読んだときに理解しやすくなります。

---

## オブジェクト指向とは

オブジェクト指向とは、現実の「モノ」をイメージしてプログラムを作る考え方です。

例えば音声なら、

* 音声というオブジェクトがある
* 音量を変える
* 保存する
* MP3へ変換する

といった機能を、そのオブジェクト自身が持っています。

Pythonでは、この考え方を使ってプログラムが作られています。

---

## 今回学んだこと

* ライブラリは便利な機能をまとめたもの
* パッケージはモジュールをまとめたもの
* モジュールはPythonファイル
* クラスは設計図
* オブジェクトはクラスから作られた実際のデータ
* メソッドはクラスやオブジェクトが持つ機能
* `.` は「〜の」「〜にある」というイメージ
* 変数名は自由だが、意味のある名前を付けると読みやすい
* Pythonはオブジェクト指向という考え方を採用している
